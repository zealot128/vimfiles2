Vimfiles
========

My vim files for Linux and Mac.


## First time install

0. Fast

    ```
    \curl -L https://raw.github.com/zealot128/vimfiles2/master/install.sh | bash
    ```


1. Manual: Clone this repository:

    ```
    $ git clone https://github.com/zealot128/vimfiles2.git ~/.vim
    cd ~/.vim
    ````



2. Symlink config

    ````
    make install
    ````

3. Install Plugins

    ````
    nvim
    :PlugInstall
    ````
